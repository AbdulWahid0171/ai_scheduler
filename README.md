## Status

This is an active side project and currently in prototype stage.

The app already supports core reminder management and AI-assisted scheduling flows, but the AI behavior is still being refined to improve consistency, correctness, and overall user experience.

## Notes

- The local model file is not included in this repository.
- Large model assets are excluded from version control.
- The source code is currently kept private while development continues.

## Roadmap

- Improve AI reliability for chat versus scheduling
- Strengthen reminder creation and update parsing
- Support multiple reminder operations in a single request
- Improve reminder editing flow
- Polish the UI and UX
- Explore future API-based architecture

## Future Direction

Possible future directions include:

- Stronger deterministic scheduling logic
- Optional API-based assistant integration
- Broader scaling beyond the current local-first prototype
